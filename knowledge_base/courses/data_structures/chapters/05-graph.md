# 图结构与图算法入门

## 学习目标

- 理解图的基本概念。
- 区分邻接矩阵和邻接表。
- 掌握 BFS 和 DFS 的基本思想。

## 核心知识点

- 顶点、边、有向图、无向图、权值。
- 邻接矩阵：适合稠密图，查询边方便。
- 邻接表：适合稀疏图，节省空间。
- BFS：借助队列逐层扩展。
- DFS：借助递归或栈深入搜索。

## 易错点

- 遍历图时忘记记录 visited，导致重复访问或死循环。
- 不清楚 BFS 和 DFS 的适用场景。
- 混淆图的存储结构和遍历算法。

## 实操案例

用邻接表表示一个校园路线图，并实现 BFS 查询从 A 楼到 B 楼的最少经过节点数。
