# 线性表、链表与顺序表

## 学习目标

- 理解线性表的逻辑结构。
- 区分顺序表和链表的存储方式。
- 能分析插入、删除、查找操作的复杂度。

## 核心知识点

- 顺序表：用连续存储空间保存元素，支持快速按下标访问。
- 单链表：每个节点保存数据和后继指针。
- 双链表：节点同时保存前驱和后继指针。
- 常见操作：头插、尾插、按位置插入、删除节点、查找节点。

## 易错点

- 链表删除节点时忘记保存前驱节点。
- 插入节点时指针赋值顺序错误。
- 误以为链表按下标访问也是 O(1)。
