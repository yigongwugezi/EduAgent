# 查找与排序

## 学习目标

- 掌握顺序查找和二分查找。
- 理解冒泡、选择、插入、快速排序的基本思想。
- 能比较常见排序算法的复杂度和稳定性。

## 核心知识点

- 顺序查找：逐个比较，适合小规模或无序数据。
- 二分查找：要求有序，时间复杂度 O(log n)。
- 冒泡排序：相邻元素比较交换。
- 插入排序：把新元素插入到已有有序区间。
- 快速排序：分治思想，选基准并划分区间。

## 易错点

- 二分查找边界条件写错。
- 快速排序递归区间划分错误。
- 只背排序复杂度，不会解释算法过程。

## 练习建议

手动模拟 `[5, 2, 9, 1, 5, 6]` 的插入排序过程，并写出每一轮数组状态。
