## 第三部分：通用组件 API

### Button

```tsx
type ButtonVariant = 'primary' | 'secondary' | 'ghost' | 'danger'
type ButtonSize = 'sm' | 'md'

<Button variant="primary" size="md" onClick={fn}>
  文字内容        // 无图标
</Button>

// primary:   bg-gray-900 text-white hover:bg-gray-800
// secondary: bg-white border border-gray-200 text-gray-700 hover:bg-gray-50
// ghost:     text-gray-600 hover:text-gray-900
// danger:    text-red-600 hover:bg-red-50
// sm:        px-3 py-1.5 text-xs
// md:        px-4 py-2 text-sm
```

### Input

```tsx
<Input
  value={val}
  onChange={fn}
  placeholder="搜索..."
  className="flex-1"
/>
// border border-gray-200 rounded-lg px-4 py-2 text-sm
// focus: ring-2 ring-brand-500/20 border-brand-500
```

### Select

```tsx
<Select value={val} onChange={fn}>
  <option>全部类型</option>
  <option>讲义</option>
</Select>
// 同 Input 样式
```

<Badge variant="default">讲义</Badge>

### 4.1 LoginPage

#### 状态定义

```
states:
  - loading:      正在读取 localStorage 学习者列表
  - ready:        已加载完成
  - creating:     正在创建新学习者（nameInput 显示中）
  - renaming:     正在重命名（nameInput + id 标识）

data:
  learners: Learner[]         从 localStorage 读取
  activeLearner: Learner | null

events:
  create(name)       → 写入 localStorage，自动选中
  switch(learner)    → 写入 activeLearner，跳转到 /
  remove(id)         → 从列表移除
  rename(id, name)   → 更新名称
  enter()            → 跳转到 /
```

#### 布局

```
整页居中:
  min-h-screen bg-gray-50 flex items-center justify-center p-6

卡片:
  w-full max-w-sm bg-white rounded-xl border border-gray-200 p-6

Logo:
  flex items-center gap-2 justify-center mb-6
  bg-brand-600 w-8 h-8 rounded-lg flex-shrink-0
  text-base font-bold text-gray-900

当前用户（activeLearner && !creating）:
  bg-gray-50 rounded-lg p-4 mb-4
  flex items-center gap-3
  头像: bg-brand-600 text-white w-10 h-10 rounded-lg text-base font-bold
  信息: 姓名(text-sm font-semibold) + ID(text-xs text-gray-400)
  进入按钮: Button variant="primary" size="sm"

分割线: Divider className="my-4"

学习者列表:
  space-y-1 max-h-48 overflow-y-auto
  每项: flex items-center gap-3 px-3 py-2.5 rounded-lg
       hover:bg-gray-50 cursor-pointer group
    头像: bg-gray-200 text-gray-600 w-8 h-8 rounded-lg text-sm font-bold
    姓名: text-sm text-gray-700
    当前标记: text-xs text-brand-600（可选）
    删除按钮: hidden group-hover:block text-gray-300 hover:text-red-500

新建按钮:
  列表底部: text-sm text-gray-500 hover:text-gray-700 py-2

新建表单（creating == true）:
  p-3 bg-gray-50 rounded-lg mt-2
  flex gap-2
  Input autoFocus maxLength={20}
  Button("创建") + Button("取消" variant="ghost")
```

#### 响应式

```
桌面/移动端: 同一布局，卡片 max-w-sm 两端自适应
```

#### 空状态

```
无学习者时：
  列表区显示: text-sm text-gray-400 text-center py-4 "还没有学习者"
  新建按钮始终可见
```
